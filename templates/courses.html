2D Animation

3D Animation

3D Animation with HKMI

3D animation is the process of creating moving images in a three-dimensional (3D) environment, giving the illusion that these digital objects are moving through a 3D space. This is done by using computer software to create and manipulate digital objects, or by using specialized hardware such as motion capture.
